kakaoOptions
: 카카오톡 메시지 옵션

  senderKey
  : 알림톡 Sender Key

  templateCode
  : 알림톡 Template Code

  buttonName
  : 카카오톡 버튼 이름, 10 자 제한

  buttonUrl
  : 카카오톡 버튼 URL, 100 자 제한

  disableSms
  : 알림톡 및 친구톡 발송에 실패하여도 문자메시지로 대체하여 발송하지 않습니다.

    입력 가능한 값

    `true`
    : 알림톡 실패시 문자로 대체하지 않음

    `false` default
    : 알림톡 실패시 문자로 대체 발송
