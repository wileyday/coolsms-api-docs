type
  : 메시지의 유형
    : 입력 가능한 값
      : `SMS`
        : 영자 90 자(한글 45 자) 이하의 단문메시지
      : `LMS`
        : 영자 2,000 자(한글 1,000 자) 이하의 장문메시지
      : `MMS`
        : 영자 2,000 자(한글 1,000 자) 이하의 텍스트와 1개의 [이미지](#) 를 포함하는 메시지
      : `ATA`
        : 한/영 포함 1,000 자의 텍스트를 발송할 수 있는 카카오톡의 [알림톡](#)
      : `CTA`
        : 한/영 포함 1,000 자의 텍스트를 보낼 수 있는 카카오톡의 [친구톡](#)
      : `PUSH`
        : 푸시메시지
